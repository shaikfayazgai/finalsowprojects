# Font Files

Place WOFF2 font files here before enabling font loading in portals and Storybook.

Required files:
- MillerDisplay-Light.woff2
- MillerDisplay-Medium.woff2
- MillerDisplay-Black.woff2
- AvenirLTStd-Light.woff2
- AvenirLTStd-Roman.woff2
- AvenirLTStd-Medium.woff2
- AvenirLTStd-Heavy.woff2
- AvenirLTStd-Black.woff2

See portal layout.tsx files for font loading code (commented out until files are placed).
See packages/ui/src/styles/storybook-fonts.css for Storybook font loading (also commented out).
